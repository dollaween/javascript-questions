# Typescript: Вопросы

---

##### 1. Какие есть базовые типы?

<details><summary><b>Ответ</b></summary>
<p>

Базовые типы:
* Boolean
* Number
* BigInt
* String
* Symbol
* Array
* Tuple
* Enum
* Unknown
* Any
* Void
* Null
* Undefined
* Never
* Object

```typescript
let bool: boolean = false
let num: number = 25
let big: bigint = 100n
let str: string = 'hello'
let list: number[] = [1, 2, 3]
let tuple: [string, number]: ['Cassie', 23]
```

</p>
</details>
